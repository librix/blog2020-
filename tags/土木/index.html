yjk优化设计

提高结构性能避免安全隐患的计算改进

1.包络设计：

  • 多塔：合塔计算和分塔计算（《高规》5.1.14）

    • 「包络设计」「结果较大值」
    • 『计算简图』『自动分塔示意』可自定义分塔
    • 「显示多塔取大构件」合塔控制的构件

  • 楼梯：考虑楼梯构件影响&不计楼梯构件影响（地震时，楼梯是一栋楼里面最安全的地方）

    • 「包络设计」「与其他模型进行包络设计」
    • 正截面，斜截面

  • 少墙框架的自动包络计算

    • 软件按框架计算时，对剪力墙的弯曲刚度折减系数。
    • 软件不能对剪力墙的轴向刚度可抗剪刚度进行折减。（机构）
    • 「设计结果」「少墙框架」

  • 多遇地震计算、大震（中震）弹性、大震（中震）不屈服设计

    • 中震不屈服设计：前处理「楼层属性」「包络设计构件」
    • 计算「只包络设计」 「显示取大」

  • 刚性连接上连接体

  1. 多模型

  • 剪力墙连梁刚度的折减，仅适用于地震效应工况；其余工况不宜折减。

    • 连梁刚度不折减组成总刚，非地震工况计算。
    • 连梁刚度折减组成总刚，地震抗震计算。

  • 整体指标（强制刚性板假定），内力、位移、配筋（非刚性板假定）

  • 强制刚性

    • 坡屋面、错层结构、开大洞口楼层、越层柱、越层支撑，按刚性楼板假定计算周期比、位移比。
    • 刚性平面：只有整体平面位移，层间杆件只有整体的平动、转动。

  • 层间突变形成的薄弱层 && 层间受剪承载力突变形成薄弱层

    • 层间受剪承载力突变形成薄弱层 构件截面配筋查看。

  • 抗震墙连梁刚度折减

    • 地震内力：连梁刚度折减
    • 位移：连梁刚度可不折减，意思是出现位移较大时。

  • 最不利地震方向地震作用

3.有限元力学模型概念

  • MPC 多点约束：避免使用惩罚函数造成的矩阵病态等问题。
  • 子空间迭代法
  • Ritz向量法：荷载空间分布，忽略不参与响应振型。【大型结构动态响应】
  • Lanczos法

结构失稳 = 屈曲（屈曲荷载、破坏形态） = 广义特征值

二阶效应系数 = 屈曲因子的倒数

比例阻尼、振型阻尼、古典阻尼矩阵、分离模态、基于应变能的阻尼比计算（抗规10.2）

  • 重力二阶效应

    • 几何刚度矩阵

      • 壳元的几何刚度矩阵（剪力墙）
      • 杆元的几何刚度矩阵
      • 空间梁元的几何刚度矩阵（一维构件：弯矩、偏心）

    • 竖向工况应力状态计算

      • yjk：竖向恒载、活载计算和组合结果
      • 竖向构件顶点质量
      • 用户指定应力状态

    • 重力二阶效应计算过程

  • 弹性连接「前处理」「节点属性」「单点约束」

    • 滑动连接

    • 滑动支座 （X平动刚度为零）

    • 减震隔震装置

    • 其他概念

      • 屈曲约束支撑
      • 阻尼器
      • 塑性单元（wen）
      • 隔震支座
      • 间隙：有效刚度、有效阻尼、刚度、间隙

  • 膜单元

    • QA4 经典膜单元

      • 带旋转自由度的精华非协调平面四边形等参
      • QA4，Allman 陈万吉、龙驭球

    • NQ6Star 改进膜单元

      • 二次完备精化非协调单元
      • 转角自由度比较完善，非规则四边形单元，无罚约束函数。
      • 温度应力计算、边框柱和剪力墙的内力协调分配，弧形墙数量较多，QA4结果不合理可以使用NQ6Star。

4.消能减震设计&隔震设计

  • 消能器：速度相关型、位移相关型，etc

  • 《抗规》5.1按照变形特征和高度决定计算方法；

  • yjk 振型分解反应谱法

    • 斜杆 = 消能器

      • 「前处理」「支撑定义」「设置减震单元」（连接属性）
      • 「节点属性」「消能器」

  • 隔震计算

    • 非线性动力时程分析

      • 常规：每一个时程积分时刻的平衡方程求解
      • yjk的快速非线性分析方法：没看懂；速度快，结果与辛普森积分接近。
      • 「节点属性」「单点约束」
      • 「弹性时程分析」FNA计算

（人工对比得出 beta）地震力水平向减震系数 beta；隔震后水平地震影响系数最大值 alpha_max1 = beta * alpha_max /phi，填入“地震影响系数最大值”，计算结果偏大。

  • 时程分析

「建模」

「双塔计算」「隔震装置」

「反应谱计算」

「FNA时程分析计算」

计算完成后，「计算参数」「地震放大系数」，导入各层放大系数到反应谱法

「直接积分法」「反应谱法计算」

  • 可根据FNA时程分析结果调整隔震支座属性。

14.节点的弹簧刚度

  • 节点约束 6 = 3+3 个自由度

  • 节点设置弹簧刚度 = 一个节点变两个节点，并间之以弹簧刚度。无需人工设置两个点。

  • 单点约束，主要用在层之间杆件的弹性约束，方式是修改上层柱的底部6个约束。

  • 「局部座标系」 x，y方向任意长线段，测量两点距离，XYZ方向增量填入。

  • 竖向刚度不起作用 = 刚度无穷大？

  • 弹性连接 、铰连接

    • 短柱，短柱与模型分开150mm，「两点连接」「弹性连接」
    • 没有空隙，短柱刚度不起作用，但是如果某自由度被释放，或有弹簧刚度时，两点的自由度会叠加短柱刚度。
    • 支座滑动方向剪力 =0；

6.优化设计节省材料的有效措施

合理的结构设计应是在满足规范要求和足够安全的前提下，最大限度地节约材料和造价，实现建筑真正的绿色和环保。

当前主要的问題是当前普遍应用的软件中对很多同题的处理存在不全面、粗放的状况，我们首先从几个方面详细梳理了现软件存在的问题，这些问题将直接导致结构钢筋、混凝土等材料的浪费和造价的增加。
