自己做一个适合自己的手机输入法

简介

手机输入的目的当然是快，一双快手自然无需考量太多无关的问题，但是总有一些“笨人”觉得搜狗输入法、百度输入法等等有各种小问题而没有办法改变，比如：

[ ] 我需要随时录入当前日期时刻。
[ ] 我需要剪切板增强软件方便自己多次复制的时候可以一次粘贴收藏。
[ ] 我有特殊的管理技巧，需要固定的格式化字符串来描述当前状态。
[ ] …

trime[1]（rime 的安卓版本）是目前非常好的一个可以自定义的输入法软件。但是需要注意的是，他的自定义属性很强，甚至不自定义简直没有办法使用。官方的文档写的细致入微，但是缺乏宏观方向，因此我想写一篇简单的备忘录，方便自己下次修改的时候不至于不知道从什么地方下手。

trime 说明

trime 有 同文输入法 和 中文输入法 两个分类，中文输入法 原为“懒人版”，拥有很多功能，必须加群才能下载。同文输入法没有配置文件（甚至无法打字），需要很多自己配置的项目，本文主要说的是中文输入法。

安装完成 APP 后，需要在设置里面看一下「用户文件夹」的路径，默认的话：

这个文件夹内的所有文件，就是控制该输入法的行为的所有文件。

用户需要的操作是：

  • 将文件放入这个文件夹内。
  • 让程序执行「部署」操作。

输入法会自动读取该文件夹内的文件。

自制输入法需要以下几个步骤，先不要急着做：

  • 新建一个文件夹，重命名为 “rime”
  • 将 tongwenfeng.trime.yaml 和 trime.yaml 复制到电脑上。
  • 新建一个空白yaml文件，如"test.trime.yaml"
  • [耗时最长、重复次数最多的一步]修改这个文件。
  • 将修改后的文件传入手机。
  • 重新部署输入法，并使用这个文件。

步骤一说明

新建一个文件夹，重命名为 “rime”。

可以选择 mkdir 命令，或者直接右键新建文件夹，或者在 totalcmd 中按 F7，没什么可说的。

步骤二说明

将 tongwenfeng.trime.yaml 和 trime.yaml 复制到电脑上。为什么是这两个文件？

当然可以选择数据线，右键复制粘贴。但是总有些麻烦，因为你可能不止一次的需要在手机、电脑来回传输文件。因此推荐使用 adb 控制，需要百度“如何安装adb”。

步骤三说明

新建一个空白yaml文件，如"test.trime.yaml"

设计输入法

想要修改一个输入法，必须先自己进行一些设计。

  • 我需要用输入法来完成什么？
  • 我需要输入法进行怎么样的展示？
  • 输入法需要哪些功能？这么功能如何利用可以满足我的需要。

我的输入法思路。

按照键盘布局：主键盘5行。
第一行：
1={1，！，日期}
2={2，@ ，时刻}
3={3，# ，ID生成器}
4={4，$ ，农历}
5={5，% ，文本编辑选择工具}
6={6，^ , 剪切板}
7={7，＆，短语常用语}
8={8，* ，中文字符}
9={9，（，英文字符}
0={0，），常用表情}

第二行：qwertyuiop 常规字母，向上滑动输入符号即可。
第三行：asdfghjkl；
a 全选 sdfg 行首行尾上页下页 hjkl 方向键（左下上右）
第四行：zxxcvbnm
z 撤销 xcv 剪切复制粘贴 bnm ，。？
shift 键 退格键
第五行：数字键盘|菜单选项|符号键盘|空格|中英文切换(功能冗余)|回车。

副键盘：

  1. 数字键盘
  2. 符号键盘
  3. 表情包键盘
  4. 特殊符号键盘

如何实现功能的准备工作

所有的代码都需要参考已经存在的trime.yaml文件书写。我参考的是 tongwenfeng.trime.yaml。

所有的代码都在编辑器中书写，我使用的是 sublime。

trime文件格式

主要修改的地方，其他直接复制：

  • 第 73 行
  • preset_keyboards
  • preset_keys （配合 preset_keyboards）

sublime 功能

  • 多行编辑
  • 正则表达式

按住ctrl-alt，再单击上下方向键 N 次，可以编辑多行。
多行编辑模式下，
- ctrl-alt-N 录入数字（需要插件）
- ctrl-左/右方向键
- alt-左/右方向键
- shift- 选中状态
- end 行尾 home 行首
其他按键，包括剪切复制粘贴等操作不再表述。

键盘格式

一个键盘总宽度是 100，每一个键的宽度是 width 定义。当 width = 10 时，意思是一排有10个键，刚好放下十个数字.
在 keys 属性中，键依次排列即可，当键的总长度大于100时，会显示在下一行。

注意 keys 不是 key。
所有的冒号后面都必须有空格，除非是 \n。

然后，我们只需要将自己需要的任何功能写入即可。
书写过程中，保持《trime 详解》的页面打开，随时 ctrl-f 查找功能键名字。
